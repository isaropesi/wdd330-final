# LitTrack - Book Discovery & Reading Tracker

A personalized book discovery and reading tracking application. Build for WDD 330 Final Project.

## Features (Week 5)
*   **Book Search**: Search for books by title, author, or ISBN using the Google Books API.
*   **Book Details**: View detailed information about a book, including synopsis, page count, and publisher.
*   **Responsive Design**: Optimized for desktop and mobile viewing with a clean, card-based layout.

## Technologies Used
*   HTML5
*   CSS3 (Custom Properties, Grid, Flexbox)
*   Vanilla JavaScript (ES Modules, Async/Await)
*   Google Books API

## Setup
1.  Clone the repository.
2.  Open `index.html` in your browser (Wait! You must use a local server due to ES Modules).
    *   **VS Code**: Right-click `index.html` -> "Open with Live Server".
    *   **Terminal**: Run `python3 -m http.server` and go to `http://localhost:8000`.
